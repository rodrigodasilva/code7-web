## Code 7 - Front-end

### Pré-requisitos

- [Ter a api rodando](https://github.com/rodrigodasilva/code7-api)
- Npm/Yarn

### Como usar

1. Clone o repositório na sua máquina

   > git clone https://github.com/rodrigodasilva/code7-web.git

2. Acesse a pasta

   > cd code7-web

3. Instale as dependências

   > yarn

4. Inicia a aplicação
   > yarn start
